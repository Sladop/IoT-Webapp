Function

The main function of this web-application is to provide an automatic mobilephone torch. The torch should be activated if the surrounding is to dark to see. This is made b reading the lightsensor of the mobile device.

HTML5 Features

Accessing the camera is a HTML5 feature. Also activating and deactivating the torch is onl possible due to the HTML5 features used in this project. The wep-application is also usable with responsive desing on a smaller mobile device.
