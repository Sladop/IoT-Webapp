Function

Ths main focus on this project is to enable the user to interact with his mobile flashlight. The web application is able to simply activate and deactivate the torch but its has also a solution for morse code implemented. With an input field you can send your own messages in morsecode thorugh your flashlight. It also has an easy access SOS button for fast alarm signals.

HTML5 Features

Accessing the camera is a HTML5 feature. Also activating and deactivating the torch is onl possible due to the HTML5 features used in this project. The wep-application is also usable with responsive design on a smaller mobile device.
